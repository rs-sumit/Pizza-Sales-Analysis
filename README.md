# Pizza-Sales-Analysis
Performed SQL-based analysis on pizza sales data to extract revenue, demand, and customer behavior insights.
– Tech Stack: SQL, MySQL, CSV Dataset
December 2025– Designed and populated relational tables for orders, order_details, pizzas, and pizza_types from raw CSV files.– Computed business metrics including total revenue, total orders, and average order value using SQL aggregations.– Analyzed sales performance by pizza category and size to identify top and low-performing products.– Evaluated daily and hourly order trends to determine peak demand periods and customer ordering patterns.– Ranked best-selling pizzas and tracked cumulative revenue trends to support data-driven business decisions.
